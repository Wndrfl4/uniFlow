# UniFlow Blog

## Описание
Разработать backend-приложение "UniFlow Blog" — университетскую блог-платформу с интегрированным AI-ассистентом и модулем управления персональными данными.

Платформа должна обеспечивать:
- публикацию и модерацию постов;
- взаимодействие пользователей с AI-ассистентом;
- кэширование AI-ответов;
- контроль лимитов запросов;
- экспорт и удаление персональных данных;
- анонимизацию профилей;
- аудит всех критически важных действий.

## Требования к коду
- Использовать только DTO во внешнем API
- SQL только через Repository
- Пароли хранить только в BCrypt
- Секреты хранить только в environment variables
- Соблюдать Java Code Conventions
- Писать чистый и читаемый код
- Добавлять обработку ошибок
- Покрывать бизнес-логику тестами

---

## Важные указания
- Всегда генерировать полный рабочий код.
- Строго придерживаться существующей архитектуры проекта.
- Не изменять структуру пакетов без необходимости.
- Перед созданием нового класса учитывать текущую реализацию.
- Использовать современные best practices Spring Boot.
- Все защищённые методы должны содержать @PreAuthorize.
- При работе с AI обязательно учитывать Redis, Kafka и лимиты ролей.
- Для локальной разработки предусмотреть mock-режим AI.
